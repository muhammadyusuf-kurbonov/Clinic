### 32Desk - dialer app

##### This app is created to identify calling person and show general information about customer

Used technologies:
Kotlin coroutines (inc Flow)
Retrofit2
Jetpack compose (+ accompanist)

Architecture is MVI

To run tests use commands:
    **For unit test**:
    `./gradlew test`

**For unit test**:
`./gradlew checkAndroidConnection`
